<a id="frontend-installation"></a>

The **OpenNebula Front-end** serves as the control plane for your entire cloud infrastructure. It is the central orchestrator responsible for deploying and managing the lifecycle of Cluster nodes, Virtual Machines (VMs), Virtual Networks, and storage datastores. Installing the Front-end is the first step in deploying an OpenNebula cloud, read this section before proceeding to [Cluster Deployment]({{% relref "software/installation_process/cluster_installation/" %}}).
